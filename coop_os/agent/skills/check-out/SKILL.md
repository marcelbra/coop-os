## Steps

1. **Review what moved today**
   - Read task files in `tasks/` with status `in-progress`
   - Ask: "What did you actually get done today?"
   - Update task statuses and add completion notes based on the answers

2. **Capture loose ends**
   - "Anything that came up today that isn't captured yet?"
   - Create task files for anything new using `tasks/template.md`

3. **Flag blockers**
   - Any task stuck? Note it in the task's description file

4. **Preview tomorrow**
   - Pull tomorrow's calendar events
   - Suggest top 3 priorities based on tasks + calendar

5. **Close**
   - One sentence on how the day went (honest, not cheerleader)
   - "Rest well." and done.
